# MelodyRoam 首页代码详细讲解

## 一、整体架构概览

首页采用经典的三层架构模式：

```
┌─────────────────────────────────────────────────────────────┐
│                    HTML 结构层                              │
│  - 导航栏 (<nav>)                                           │
│  - 封面墙容器 (<div class="cover-wall">)                    │
│    - 静态背景封面 (<div class="static-covers">)             │
│    - 动态封面入口 (<div class="dynamic-covers">)            │
│    - 底部文字 (<div class="wall-text">)                     │
├─────────────────────────────────────────────────────────────┤
│                    CSS 样式层                               │
│  - 全局重置样式 (*)                                         │
│  - 导航栏样式 (.navbar)                                    │
│  - 封面墙样式 (.cover-wall, .static-cover, .dynamic-cover) │
│  - 响应式布局 (@media)                                     │
├─────────────────────────────────────────────────────────────┤
│                    JavaScript 交互层                        │
│  - 鼠标跟随效果 (mousemove)                                 │
│  - 背景散开效果 (mouseleave)                                │
│  - 页面加载动画 (load)                                      │
└─────────────────────────────────────────────────────────────┘
```

## 二、HTML 结构详解

### 2.1 基本结构

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>MelodyRoam - 音序漫游</title>
  <link rel="stylesheet" href="./css/common.css">  <!-- 引入样式文件 -->
</head>
<body>
  <!-- 页面内容 -->
  <script src="./js/common.js"></script>  <!-- 引入交互脚本 -->
</body>
</html>
```

**关键点：**
- `viewport` 设置确保移动端适配
- CSS 和 JS 文件在页面末尾引入，优化加载性能

### 2.2 导航栏结构

```html
<nav class="navbar">
  <div class="logo">MelodyRoam 音序漫游</div>
  <div class="nav-links">
    <a href="index.html">首页</a>
    <a href="hiphop.html">HipHop</a>
    <a href="rnb.html">R&B</a>
    <a href="gufeng.html">古风</a>
    <a href="kpop.html">K-Pop</a>
  </div>
</nav>
```

**设计思路：**
- 简洁的导航结构，包含 Logo 和 5 个导航链接
- 使用 `<a>` 标签实现页面跳转

### 2.3 封面墙结构

```html
<div class="cover-wall">
```

**层级设计：**

## 三、CSS 样式详解

### 3.1 全局重置样式

```css
* {
  margin: 0;

body {
  background: #f5f5f7;
  overflow-x: hidden;  /* 隐藏水平滚动条 */
}
```

### 3.2 导航栏样式（核心亮点：毛玻璃效果）

```css
.navbar {
  position: fixed;        /* 固定定位，始终显示在顶部 */
  top: 0;
  left: 0;
  width: 100%;
  height: 72px;
  padding: 0 70px;
  background: rgba(255, 255, 255, 0.35);  /* 半透明背景 */
  backdrop-filter: blur(20px);              /* 毛玻璃模糊效果 */
  -webkit-backdrop-filter: blur(20px);     /* Safari 兼容 */
  display: flex;
  justify-content: space-between;  /* Logo 左，导航右 */
  align-items: center;             /* 垂直居中 */
  z-index: 999;                   /* 最高层级 */
}
```

**毛玻璃效果原理：**
- `backdrop-filter: blur(20px)` 对元素后方内容进行模糊处理
- `rgba(255, 255, 255, 0.35)` 提供半透明底色
- 组合产生现代感的毛玻璃视觉效果

### 3.3 导航链接悬停动画

```css
.nav-links a::after {
  content: '';                 /* 伪元素生成下划线 */
  position: absolute;
  left: 0;
  bottom: -8px;
  width: 0;                    /* 初始宽度为 0 */
  height: 3px;
  border-radius: 3px;
  background: #1d1d1f;
  transition: width 0.3s ease; /* 宽度过渡动画 */
}

.nav-links a:hover::after {
  width: 100%;                 /* 悬停时展开 */
}
```

**动画实现原理：**
- 使用 CSS 伪元素 `::after` 创建下划线
- 通过 `width` 属性从 0 → 100% 实现展开动画
- `transition` 控制动画时长和缓动效果

## 四、JavaScript 交互详解

### 4.1 鼠标跟随效果（核心功能）

```javascript
document.querySelector('.cover-wall').addEventListener('mousemove', (e) => {
  // 1. 计算鼠标偏移量（归一化处理）
  const x = (e.clientX / window.innerWidth - 0.5) * 20;
  const y = (e.clientY / window.innerHeight - 0.5) * 20;

  // 2. 主封面跟随鼠标移动
  const mainCover = document.querySelector('.main-cover');
  mainCover.style.transform = `translate(calc(-50% + ${x}px), calc(-50% + ${y}px))`;

  // 3. 副封面反向移动（视差效果）
  const subCovers = document.querySelectorAll('.sub-cover');
  subCovers.forEach((cover, index) => {
    const reverseX = x * (0.5 - index * 0.2);  // 系数递减
    const reverseY = y * (0.5 - index * 0.2);
    cover.style.transform = `translate(${reverseX}px, ${reverseY}px)`;
  });

  // 4. 背景封面散开效果
  const mouseX = e.clientX;
  const items = document.querySelectorAll('.static-cover');
  items.forEach(item => {
    let rect = item.getBoundingClientRect();
    let itemCenter = rect.left + rect.width / 2;
    let offset = (itemCenter - mouseX) * 0.08;  // 距离产生斥力
    item.style.transform = `translateX(${offset}px)`;
  });
});
```

**核心算法解析：**

| 步骤 | 变量 | 计算逻辑 | 效果 |
|-----|------|---------|------|
| 1 | x, y | `(鼠标位置/窗口宽度 - 0.5) * 20` | 归一化到 -10 ~ 10 |
| 2 | mainCover | `translate(calc(-50% + x), calc(-50% + y))` | 同向跟随 |
| 3 | subCovers | `x * (0.5 - index * 0.2)` | 反向视差，幅度递减 |
| 4 | static-cover | `(封面中心 - 鼠标X) * 0.08` | 距离越远偏移越大 |

**视差效果原理：**
- 主封面与鼠标同向移动，增强跟随感
- 副封面反向移动，产生层次深度感
- 背景封面根据距离散开，模拟物理斥力

### 4.2 鼠标离开恢复

```javascript
document.querySelector('.cover-wall').addEventListener('mouseleave', () => {
  document.querySelectorAll('.static-cover').forEach(item => {
    item.style.transform = 'translateX(0)';  // 重置位置
  });
});
```

### 4.3 页面加载动画

```javascript
window.addEventListener('load', () => {
  // 背景封面渐显
  document.querySelectorAll('.static-cover').forEach(cover => {
    cover.style.opacity = 1;
  });

  // 动态封面依次渐入（带延迟）
  const dynamicCovers = document.querySelectorAll('.dynamic-cover');
  dynamicCovers.forEach((cover, index) => {
    cover.style.opacity = 0;           // 初始透明
    cover.style.transform += ' scale(0.8)';  // 初始缩小
    
    setTimeout(() => {
      cover.style.opacity = 1;         // 渐显
      cover.style.transform = cover.style.transform.replace(' scale(0.8)', '') + ' scale(1)';  // 放大
    }, 400 + index * 300);             // 每个间隔 300ms
  });
});
```

**动画时序：**
| 封面 | 延迟时间 | 效果 |
|-----|---------|------|
| 第1个 | 400ms | 最先出现 |
| 第2个 | 700ms | 延迟 300ms |
| 第3个 | 1000ms | 延迟 600ms |
| 第4个 | 1300ms | 最后出现 |

## 五、代码优化建议

### 5.1 性能优化

**问题：** 41 张背景图片全部同时加载，可能影响首屏性能

**优化方案：**
```javascript
// 图片懒加载
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      const img = entry.target;
      img.style.backgroundImage = `url(${img.dataset.src})`;
      observer.unobserve(img);
    }
  });
}, { threshold: 0.1 });

document.querySelectorAll('.static-cover').forEach(cover => {
  observer.observe(cover);
});
```

**问题：** JavaScript 代码缺乏模块化

**优化方案：**

## 六、总结

首页代码实现了以下核心功能：

| 功能模块 | 实现方式 | 技术亮点 |
|---------|---------|---------|
| 导航栏 | CSS 固定定位 + backdrop-filter | 毛玻璃效果 |
| 背景封面 | CSS 绝对定位 + 百分比布局 | 散落视觉效果 |
| 动态封面 | CSS transform + z-index | 层级管理 |
| 鼠标跟随 | JavaScript mousemove + 距离算法 | 视差效果 |
| 加载动画 | setTimeout + CSS transition | 依次渐入 |
| 响应式 | CSS Media Queries | 移动端适配 |

代码结构清晰，使用原生技术栈实现了丰富的交互效果，是一个高质量的前端页面实现。
