🎵 MelodyRoam 音序漫游
================

一个动静结合的音乐分类展示网站，包含 HipHop、R&B、古风、K-Pop 四大音乐分类。

📖 项目简介
----------------

MelodyRoam（音序漫游）是一个动静结合的音乐分类展示网站，包含 HipHop、R&B、古风、K-Pop 四大音乐分类。

🎨 "静" - 精心设计的视觉布局和内容展示
- 语义化的 HTML 结构
- 美观的 CSS 样式设计
- 各页面独特的主题色彩风格

✨ "动" - 丰富的交互效果和动画体验
- 鼠标视差跟随效果
- 背景封面散开动画
- 动态渐变背景
- 自定义光标跟随
- 卡片悬浮动画
- 加载淡入序列动画
- 波形播放指示器
- 标签筛选交互

🎯 项目特色
----------------

- 🌌 **沉浸式封面墙** - 41张专辑封面散落分布，营造音乐氛围
- 🖱️ **鼠标视差效果** - 封面随鼠标移动产生3D空间感
- 🔮 **毛玻璃导航栏** - 现代UI设计，半透明磨砂效果
- 🎨 **多样风格展示** - 每个音乐分类都有独特的设计风格
- ⚡ **流畅动画过渡** - CSS3动画实现丝滑用户体验

📂 项目结构
----------------

- 📁 MelodyRoam/

    - 📄 hiphop.css            HipHop 页面样式

    - 📄 gufeng.css            古风 页面样式

  - 📁 js/                     交互脚本

🚀 快速开始
----------------

📦 本地运行

1. 克隆项目
   ```bash
   git clone https://gitee.com/你的用户名/MelodyRoam.git
   ```

2. 启动服务
   - 方法一：直接用浏览器打开 index.html
   - 方法二：使用 VS Code 的 Live Server 插件
   - 方法三：使用 Python 简易服务器
     ```bash
     python -m http.server 8080
     ```
     然后访问 http://localhost:8080

3. 访问首页
   打开浏览器访问 http://localhost:8080

📄 部署指南
----------------

🌐 Gitee Pages 部署

1. 将项目上传至 Gitee 仓库
2. 进入仓库 → 服务 → Gitee Pages
3. 选择 master 分支
4. 点击启动
5. 获得在线访问链接

☁️ Vercel 部署

1. 将项目上传至 GitHub
2. 登录 Vercel 并导入仓库
3. 点击 Deploy
4. 获得在线访问链接

📋 开发规范
----------------

📝 文件命名
- HTML 文件：小写字母 + 连字符（如 hiphop.html）
- CSS/JS 文件：与页面名称对应

📍 路径规范
- 根目录页面引用 CSS：css/xxx.css
- pages 子目录页面引用 CSS：../css/xxx.css

💻 代码风格
- 使用语义化 HTML 标签
- CSS 类名使用小写字母 + 连字符
- JavaScript 使用 ES6+ 语法
